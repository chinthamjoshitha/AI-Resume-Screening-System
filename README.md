# AI Resume Screening System

## Overview
AI-powered Resume Screening System that analyzes resumes against job descriptions and calculates match scores.

## Features
- Upload Resume PDF
- Extract Resume Text
- Match Resume with Job Description
- Calculate Resume Match Score
- Display Missing Skills

## Technologies Used
- Python
- Streamlit
- PyPDF2
- Pandas
